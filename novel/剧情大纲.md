# core

## topic

- **乙女 (恋爱游戏)**
- **恐怖 (恐怖真相)**
- **合成 (资源获取)**
- **文字游戏 (推进剧情，获得情绪，了解结局)**
- **背景**：太子集团的盘总在缅北经营毒品诈骗生意，洗钱炒币哥是他的主要洗钱工具，而超级打手则是他身边的保镖。玩家以太子集团合作伙伴的审计员身份进入这个世界。

## timeline

> 园区围攻是全局分水岭事件，由三条情报线同时成熟触发：打手长期积累的卧底证据、洗币哥链上资金异常被监控、女主完成最终情报传送。三线缺一不可。

### 主时间轴

```
T-90天  【入场】玩家以审计员身份进入园区
        └── 接触四个角色，各自故事线开始

T-60天  【各线展开】
        ├── 盘总开始对玩家动真心，内部启动转型计划
        ├── 打手卧底身份开始受到怀疑，被盘总带在身边贴身监视
        ├── 洗币哥察觉园区资金链异常，开始动摇
        └── 老鸨子出现第一次精神不稳定信号

T-30天  【内部崩溃节点】——全局最重要的隐藏节点
        ├── 盘总发现女主卧底身份，选择沉默，烧掉调查报告
        │   当晚在女主提过的那个城市名下登记了一套房
        ├── 打手被调至盘总身边，两人心知肚明，谁都没挑破
        ├── 洗币哥接到一个电话，挂断后删掉手机所有照片
        └── 老鸨子第一次在工作中公开崩溃，原因不明

        ※ 玩家只能在自己所在的故事线看见这一天的局部
        ※ 打通全部四条线后解锁隐藏成就「那三十天」，拼出完整真相

T-7天   【情报传出】
        ├── 女主完成最后一批情报传送
        ├── 打手从内部配合，调整园区防线布局
        └── 洗币哥在围攻前48小时携资产消失

T-0     【园区围攻】——各结局在此分叉
        ├── 政府武装部队攻入园区
        ├── 盘总在办公室独自等待，不抵抗，自杀（主线悲剧结局）
        ├── 打手在大门处从内部打开防线
        ├── 洗币哥已在另一个城市，看新闻
        └── 女主在撤离车上收到盘总最后一条消息

T+数天  【余震开始】
T+数月  【延后结局收尾】
T+数年  【时间跳跃结局】
```

### 各角色在T-0的位置

| ------ | ---------- | -------------------------------- |
| 盘总   | 办公室     | 独自等待，不抵抗，自杀           |
| 打手   | 园区大门   | 从内部打开防线配合部队           |
| 洗币哥 | 另一个城市 | 打开新闻，三个字："盘总死了"     |

### 结局时间类型

**T-0之前结束**——玩家提前离场，围攻只是后来听说的事，情感基调：轻，带一点遗憾

- 老鸨子·及时止损5
- 洗币哥·小小损失1（T-45天前后离场）
- 各线·核心玩家结局

**T-0作为高潮**——围攻事件本身是结局的一部分，情感基调：紧绷，史诗感

- 盘总·高傲的盘总
- 打手·最好的未来4
- 打手·恶女的权杖3
- 洗币哥·发现异常3
- 老鸨子·正义的反击4

**T-0之后收尾**——围攻结束了，故事还没完，情感基调：沉，余震比爆炸更难受

- 盘总·陷入牢狱1（T+数月，监狱探视）
- 盘总·狗肉的朋友2（T+数周，法庭宣判）
- 打手·生离死别1（T+数天，葬礼）
- 洗币哥·黑吃黑4（T+不确定，陌生城市，开放结局）
- 老鸨子·被骗网络借体2（T+数年，时间跳跃）

## world

### 常规世界

- **主角**: 妖艳老鸨子
- **配角**: 霸道盘总、超级打手
- **故事**: 精神不稳定，玩家进入园区看到的第一处可疑资产。。。
- **可能结局**:
  - **被骗1**: 玩家陷入甜蜜陷阱，被诈骗倾家荡产，网贷还不清，个人信息被泄露。（T-0之前收尾）
  - **被骗网络借体2**: 感化老鸨子失败，婚后对方重操旧业，孩子无法考公且自己感染艾滋。（T+数年）
  - **妖艳老鸨子的狗链3**: 玩家通过手段换掉老鸨子的药，彻底反向控制了老鸨子。（T-0前后）
  - **正义的反击4**: 发现异常后及时止损，最终协助警方破获诈骗团伙。（T-0高潮）
  - **及时止损5**: 察觉不对及时退出。（T-0之前）
  - **核心玩家6**: 对该角色失去兴趣，中断并进入别的世界。

### 黑暗世界

- **主角**: 霸道盘总
- **配角**: 洗钱炒币哥、超级打手
- **故事**: 缅甸华人黑帮老大的养子，从小在园区长大，18岁接手园区。嬴政式人格：强控制欲、绝对理性、深处孤独。清楚地知道诈骗无论伦理还是发展都是死路，一直想转型洗白，但整个系统不允许他停下来。
- **可能结局**:
  - **陷入牢狱1**: 共犯之爱——玩家爱上了盘总，一步步被卷进去。园区被围攻，盘总被抓，没有供出玩家。女主在法庭上为转型意图作证。他坐了3年牢，出来之后去了女主当初提过的那个城市，他们在街头相遇。（T+数年）
  - **狗肉的朋友2**: 女主是C国卧底，与同为卧底的打手配合窃取证据。盘总对玩家动了真心，开始制定转型计划，甚至签下第一步的文件。女主看见那份文件，选择无视。最终将盘总绳之以法，坐上他的位置后在抽屉里发现写着那个城市名字的便签。（T+数周）
  - **原乡的味道3**: 女主真的感化了盘总一部分，但手下不允许这件事发生。最后一次通话他说"我来接你"，她等了很久，他没来——他自己也是园区的囚徒，只是笼子更大。女主被卖往L国。（T-0之前）
  - **高傲的盘总4**: 盘总发现了女主卧底身份，选择沉默。他提前把核心情报放在女主能找到的地方，调离最忠心的手下，帮她完成任务。园区被攻入前，他坐在办公室，桌上放着一杯酒和女主第一次来时落下的某个小物件，给她发了最后一条消息，然后开枪自杀。女主在撤离车上看见消息，把手机扣过去，没有让任何人看见她的表情。（T-0高潮）
  - **核心玩家5**: 被其他角色吸引走。

### 反转世界

- **主角**: 超级打手
- **配角**: 洗钱炒币哥、妖艳老鸨子
- **故事**: 名义上是园区安保负责人，实际上是盘总唯一信任的贴身保镖。想见盘总先过他这关。真实身份是C国卧底，负责监视盘总并收集证据。在盘总身边待得足够久，足够清楚盘总想转型不是假的，足够清楚盘总对女主的感情是真的——但他还是执行了任务。
- **可能结局**:
  - **生离死别1**: 最终只见到对方的遗照与葬礼。（T+数天）
  - **恶念四起2**: 走向双双堕落的结局。（因审核风险暂留白）
  - **恶女的权杖3**: 园区被围后渗透并肢解盘总势力，最终在海岛别墅看到那个背影。（T-0高潮后）
  - **最好的未来4**: 两人在不知道对方是卧底的情况下产生了真实的情感，玩家与打手地下党接头互认卧底身份，T-0时打手从内部打开防线，任务完成。最终两人结婚，过上平静生活——他知道那个平静是用什么换来的。（T-0高潮，T+收尾）
  - **核心玩家5**: 被其他角色吸引走。

## story

### 堕落世界

- **主角**: 洗钱炒币哥
- **配角**: 妖艳老鸨子、霸道盘总
- **故事**: 园区财务总管，玩家打交道最久的人。落魄的前基金经理兼加密交易所前合规官，因投资失败欠下巨额债务，掌握洗钱链路所有漏洞，在盘总引诱下开始设计资金通道，逐渐无法自拔。住所很好但空荡荡，随时像在准备跑路。
- **可能结局**:
  - **小小损失1**: 察觉链路异常，关系没深入就切断。他发现找不到玩家，发了一条消息："聪明。"很久后玩家在新闻画面里看见一个低着头被带走的人，穿着很好，看不清脸。（T-45天前离场，T-0后远观）
  - **大大损失2**: 玩家太信任他，财务状况全部暴露，被拿捏后账户被用于走账进入监控名单。园区围攻前48小时他已携资产消失。玩家正在应付警方调查时看见围攻新闻，意识到自己从始至终只是他的一项资产。（T-0时玩家在接受调查）
  - **发现异常3**: 系统搜集链上证据配合警方。他察觉了，但没有对玩家动手——可能他在等一个被抓的理由。公寓里被捕，没有反抗。法庭上他看了玩家一眼，然后看向别处。庭审后玩家收到看守所寄来的信，只有一句话。（T-0高潮，T+数月收尾）
  - **黑吃黑4**: 玩家发现他掌握的洗钱链路数据本身就值钱，反向接近控制。拿到全部筹码后发现体检报告——长期高压饮酒睡眠剥夺，身体状况很差。围攻发生时两人已在另一个城市。他打开新闻，三个字："盘总死了"，然后去倒了杯水。结局画面：两人坐在窗边，他的药放在桌上，谁都没有说话。（T-0时不在场，开放结局）
  - **核心玩家5**: 被其他角色吸引走。

## logic

### characters

- **洗钱炒币哥**: 金融博士出身，前基金经理+前加密交易所合规官，如《绝命毒师》般误入歧途。自由意志最强的人，留下来是因为他选择留下来。
- **霸道盘总**: 缅甸华人黑帮老大的养子，从小在园区长大，18岁接手。嬴政式人格。知道犯罪是死路，一直想转型，但系统不放他走。
- **妖艳老鸨子**: 原本清白的男子因变故沦落风尘，精神状态不稳定，是玩家进入园区看见的第一个人。
- **超级打手**: C国卧底，名义安保负责人，实际贴身保镖。在盘总身边待太久，清楚这个人本来可以不是罪犯，还是执行了任务。

### 隐藏机制

- **「那三十天」成就**：打通全部四条故事线后解锁，拼出T-30天各角色的真实状态，看见完整的内部崩溃节点。
- **结局时间感知**：结局页面不显示日期，用环境细节暗示时间距离。例：「烟还没散」→ T+2天；「他的案子开庭了」→ T+90天；「你已经不记得他长什么样了」→ T+数年。
