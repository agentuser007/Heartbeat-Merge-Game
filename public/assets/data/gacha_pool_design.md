# 《心动合成》抽卡池完整设计文档

> 本文档定义了心动扭蛋机全部 100 个池项、8 张 SSR·CG 回忆卡的 32 段剧情，
> 以及碎片/百搭牌/剪刀/保底等核心规则。作为后续编码的数据蓝本。

### CG 1：🌸 青涩初吻（绑定 `ssr_gen_lip_6` · 唇妆Lv.6 · 林墨白）

**第1段：走廊擦肩**
午后的走廊空无一人，我抱着课本匆匆赶往教室。转角处，一堵温暖的"墙"挡住了去路——课本散落一地，我抬头，对上了林墨白那双幽深的眼睛。"走路不看路？"他冷冷地说，却弯腰帮我捡起了每一本书。指尖触碰的瞬间，我的心跳漏了一拍。

**第2段：天台独处**
追着跑丢的笔记来到天台，发现林墨白正独自靠在栏杆上。夕阳将他冷峻的侧脸镀上了一层柔和的金色。他注意到我，罕见地没有转身离开。"……过来坐。"他拍了身旁的位置。那是我第一次看到他表情里的柔软，晚风吹起他的发梢，我的心在寂静中剧烈跳动。

**第3段：雨中共伞**
放学突降暴雨，我站在校门口手足无措。一把黑色雨伞无声地撑到我头顶。"别感冒。"林墨白面无表情地说着，伞却大半倾斜在我这边。雨声如鼓，我们的肩膀几乎贴在一起。走到岔路口时，他突然停下："……明天，我送你。"那不是询问，是陈述。

**第4段：星空告白**
天台上，林墨白罕见地主动约我。他指着远处："你看那颗星。"我顺着他的手看去，却被他轻轻扳过脸。"我在说近的。"他的声音低沉，第一次带着温度。星空下，他缓缓靠近，额头相抵。"从走廊那天起……你就是我唯一的意外。"他的唇，带着薄荷的微凉，轻轻落在我的唇上。

毕业那天，陆之昂在操场上支起了一张长桌——上面摆满了他亲手做的菜。每道菜旁都贴着一张手写卡片：第一次做给你的排骨、你受伤时的热可可、祭典上的苹果糖……最后一道菜前，他单膝跪地，打开一个戒指盒。"我用所有味道向你承诺，"他眼眶泛红却笑得灿烂，"余生的每一餐，我都陪你吃。"

## 七、百搭牌规则

1. **获取：** SR级抽卡获得，存入背包消耗品栏
2. **使用方式：** 从背包拖出百搭牌到棋盘上任意 Lv.1-6 普通物品上
3. **效果：** 目标物品无条件升1级（如 lip_3 → lip_4）
4. **限制：**
   - 不可对 Lv.7 和 Lv.8 物品使用（弹窗提示"等级过高，百搭牌无法使用"）
   - 不可对生成器使用（弹窗提示"生成器无法使用百搭牌"）
   - 升级后的物品不触发合并动画，直接替换
5. **特殊百搭牌附加效果：**
   - 签名版：额外恢复2体力
   - 幸运版：30%概率再升1级（不超过Lv.7）
   - 双子版：相邻1个同类物品也升1级

## 八、时空剪刀规则

1. **获取：** SR级抽卡获得，存入背包消耗品栏
2. **使用方式：** 在背包中激活剪刀 → 点击棋盘上目标物品
3. **效果：** 目标物品消失 → 在原位及相邻空位生成2个低1级的同类物品
4. **限制：**
   - 不可对 Lv.1 物品使用（无法再拆分）
   - 不可对生成器使用
   - 需要至少1个相邻空位（三重剪刀需要2个）
5. **防循环机制：** 剪刀产出的物品标记"被剪"状态
   - 被剪物品不可再被时空剪刀拆分
   - 被剪物品被合并后，产出的新物品清除"被剪"标记
6. **特殊剪刀附加效果：**
   - 签名版·冷静：产物25%概率升1级
   - 签名版·暴力：不要求相邻空位（溢出物品放任意空位）
   - 签名版·浪漫：额外产出1个小瓶体力药水
   - 签名版·裁决：产物跳过"被剪"标记
   - 幸运版：产物30%概率各升1级
   - 三重版：拆分为3个而非2个

## 九、保底系统设计

### 9.1 小保底（50抽）

- 每次抽卡（无论单抽还是十连中的每一抽）计数器 +1
- 计数器达到 **50** 时，下一次抽卡 **必定** 获得随机 SSR
- 触发小保底后计数器 **不重置**，继续向大保底累积

### 9.2 大保底（100抽）

- 计数器达到 **100** 时，触发"心动自选"
- 弹出选择面板，玩家可从 8 个 SSR 中 **指定** 获取1个
- 触发大保底后计数器 **重置为 0**

### 9.3 保底计数器规则

| 事件              | 计数器变化         |
| :---------------- | :----------------- |
| 单抽              | +1                 |
| 十连抽            | +10                |
| 抽到SSR（非保底） | 不重置，继续累积   |
| 触发小保底        | 不重置，继续累积   |
| 触发大保底        | 重置为0            |
| 每日0点           | 不重置（永久累积） |

### 9.4 保底UI显示

- 抽卡面板显示当前保底计数：`距离小保底：XX/50 | 距离大保底：XX/100`
- 小保底进度条（蓝色），大保底进度条（金色）
- 大保底达到80抽后，进度条开始闪烁提醒

## 十一、回收返还体力表

> 替代原有"卖金币"机制，双击棋盘物品改为回收返体力。

| 物品等级 | 真实制造成本（体力） | 回收返还体力 | 亏损 | 设计意图               |
| :------- | :------------------- | :----------- | :--- | :--------------------- |
| Lv.1     | 1                    | 0            | 1    | 惩罚无脑点击，逼迫规划 |
| Lv.2     | 2                    | 1            | 1    | 基础止损               |
| Lv.3     | 4                    | 3            | 1    | 鼓励多合一步再卖       |
| Lv.4     | 8                    | 6            | 2    | 中期续航               |
| Lv.5     | 16                   | 12           | 4    | 高阶心痛补偿           |
| Lv.6     | 32                   | 24           | 8    | 极少回收，鼓励使用     |
| Lv.7     | 64                   | 48           | 16   | 几乎不会回收           |
| Lv.8     | 128                  | 96           | 32   | 纪念品，不鼓励回收     |

**核心公式验证：** 合成后回收 > 分别回收之和

- 2×Lv.2 各卖1 = 2，合成Lv.3卖3 = 3 → 赚1 ✓
- 2×Lv.3 各卖3 = 6，合成Lv.4卖6 = 6 → 持平 ✓
- 2×Lv.4 各卖6 = 12，合成Lv.5卖12 = 12 → 持平 ✓
- 2×Lv.5 各卖12 = 24，合成Lv.6卖24 = 24 → 持平 ✓

> Lv.3以上"合成再卖"至少不亏，Lv.2→Lv.3是唯一"赚"的节点，激励玩家至少合到Lv.3。
